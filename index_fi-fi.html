Linuxbrew

Homebrew pakettihallinta Linuxille

Näytä GitHubissa Lataa .zip Lataa .tar.gz

Linuxbrew on haara Homebrewistä, Mac OS-pakettihallinnasta Linuxille.

Se voidaan asentaa kotikansioosi, eikä se tarvitse pääkäyttäjän oikeuksia. Samaa pakettienhallintaa voi käyttää Linux-palvelimessasi, sekä Mac-läppärissäsi. Modernin version asentaminen glibcistä ja gccstä in kotikansioosi vanhassa jakelussa kestää noin viisi minuuttia.

Ominaisuudet, käyttö ja asennusohjeet on koottu kotisivuille.

Saadaksesi päivityksiä suurimmista muutoksista, tilaa Linuxbrew-päivitykset GitHubissa.

Asenna Linuxbrew (tl;dr)

Liitä tämä komentokehotteeseen:

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Linuxbrew/install/master/install)"
PATH="$HOME/.linuxbrew/bin:$PATH"

Muokkaa tiedostoa ~/.bash_profile ja lisää ~/.linuxbrew/binPATHiisi:

echo 'export PATH="$HOME/.linuxbrew/bin:$PATH"' >>~/.bash_profile

Olet valmis! Kokeile asentaa paketti seuraavalla komennolla:

brew install hello

Käytä komentoa brew doctor selvittääksesi yleisimmät viat.

Katso Järjestelmävaatimukset ja riippuvuudet sekä Asennus alla saadaksesi lisätietoja.

Ominaisuudet

  • Can install software to a home directory and so does not require sudo
  • Install software not packaged by the native distribution
  • Install up-to-date versions of software when the native distribution is old
  • Use the same package manager to manage both your Mac and Linux machines

Järjestelmävaatimukset ja riippuvuudet

  • Ruby 1.8.6 tai uudempi
  • GCC 4.4 tai uudempi
  • Linux 2.6.16 tai uudempi
  • 64-bittinen x86 tai 32-bittinen ARM (Raspberry Pi)

Liitä tämä komentokehotteeseen:

Debian tai Ubuntu

sudo apt-get install build-essential curl git python-setuptools ruby

Fedora, CentOS tai Red Hat

sudo yum groupinstall 'Development Tools' && sudo yum install curl git irb python-setuptools ruby

32-bit x86-alustat

Linuxbrew ei tällä hetkellä tue 32-bit x86-alustoja. It would be possible for Linuxbrew to work on 32-bit x86 platforms with some effort. Pull requests would be welcome if someone were to volunteer to maintain the 32-bit x86 support.

Pullot

Pullot ovat Linuxbrewin ennaltakäännettyjä binääripaketteja. Linuxbrew-pullot toimivat missä tahansa Linux-järjestelmässä. Tosin ne vaativat glibc 2.19 tai uudemman. Järjestelmissä, joissa on vanhempi versio glibcstä, Linuxbrew asentaa uusimman version glibcstä, kun asennat pullotetun kaavan. Jos haluat käyttää glibctä, jonka on toimittanut järjestelmäsi, ja haluat kääntää kaikki kaavat lähdekoodista, lisää sinun .bashrc- tai .zshrc-tiedostoon seuraava:

export HOMEBREW_BUILD_FROM_SOURCE=1

Asennus

Liitä tämä komentokehotteeseen:

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Linuxbrew/install/master/install)"

Tai jos haluat:

git clone https://github.com/Linuxbrew/brew.git ~/.linuxbrew

Lisää tämä tiedostoon .bashrc tai .zshrc:

export PATH="$HOME/.linuxbrew/bin:$PATH"
export MANPATH="$HOME/.linuxbrew/share/man:$MANPATH"
export INFOPATH="$HOME/.linuxbrew/share/info:$INFOPATH"

Olet valmis! Kokeile asentaa paketti seuraavalla komennolla:

brew install hello

Jos käytät vanhempaa jakeluversiota Linuxista, ensimmäisen paketin asentaminen asentaa myös uusimman version gccsta.

What Packages Are Available?

  1. Kirjoita brew search saadaksesi luettelon.
  2. Tai vieraile braumeister.org selataksesi paketteja verkossa.
  3. Tai käytä brew search --desc <keyword> selataksesi paketteja komentorivillä.

Laajempi dokumentaatio

brew help, man brew tai vilkaise meidän dokumentaatiota.

Ongelmien kartoittaminen

Ensin, ajathan brew update ja brew doctor komentokehotteessa.

Toiseksi, lue Viankartoittamisen tarkistuslista.

Jos et lue näitä, se kestää paljon kauemmin meiltä auttaa ongelmasi kanssa.

Jokin meni rikki!

Many of the Homebrew formulae work on either Mac or Linux without changes, but some formulae will need to be adapted for Linux. If a formula doesn't work, open an issue on GitHub or, even better, submit a pull request.

Contributing

We'd love you to contribute to Linuxbrew or its upstream, Homebrew. Ensin, luethan meidän Contribution Guide sekä Code of Conduct.

We explicitly welcome contributions from people who have never contributed to open-source before: we were all beginners once! We can help build on a partially working pull request with the aim of getting it merged. We are also actively seeking to diversify our contributors and especially welcome contributions from women from all backgrounds and people of colour.

Please see our guidelines on whether to send pull requests to Linuxbrew or Homebrew.

A good starting point for contributing is running brew audit (or brew audit --strict) with some of the packages you use (e.g. brew audit wget if you use wget) and then read through the warnings, try to fix them until brew audit shows no results and submit a pull request. If no formulae you use have warnings you can run brew audit without arguments to have it run on all packages and pick one. Good luck!
